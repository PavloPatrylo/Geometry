# Geometry
### Лабораторні роботи з обчислювальної геометрії
**Лабораторні** - Святослав Златоус

**Лекції** - Петро Венгерський

Лабораторна робота 1 - Розробити анімаційну програму, яка виконує кольорову анімацію набору різнокольорових концентричних кіл так, що за рахунок зміни кольорів та плавної зміни розмірів кіл створити ілюзію руху.

Лабораторна робота 2 - Реалізувати  алгоритм  побудови   інтерполюючої   поверхні Q(u,v). Результати представити у вигляді проекцій на площини  x=0, y=0, z=0 сімейства координатних кривих Q(u,v ) довільні криві, Q(u,v) - лінійна поверхня Кунса

ЛАбраторна робота 3 - Реалізувати алгоритм відсікання відрізків двовимірним прямокутним вікном. Алгоритм має на першій ітерації визначати та зображати повністю видимі відрізки; а також визначати і відкидати повністю невидимі відрізки. Розробити алгоритм відсікання відрізків кількома вікнами, якщо вікна мають пріоритет.
-простий алгоритм двовимірного відсікання
-алгоритм Сазерленда-Коена
-алгоритм розбиття середньою точкою.
